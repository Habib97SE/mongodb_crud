# MongoDB  CRUD

## MongoDB CRUD Operations 

insertEmployee() - Insert a new employee record into the database

insertCustomer() - Insert a new customer record into the database

readEmployee() - Read an employee record from the database

readCustomer() - Read a customer record from the database

readAllEmployees() - Read all employee records from the database

readAllCustomers() - Read all customer records from the database

updateEmployee() - Update an employee record in the database

updateCustomer() - Update a customer record in the database

deleteEmployee() - Delete an employee record from the database

deleteCustomer() - Delete a customer record from the database
